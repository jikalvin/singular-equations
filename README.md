# singular-equations

## How to 

[-] Clone repository
`git clone https://github.com/jikalvin/singular-equations.git`

[-] Access cloned repository
`cd singular-equations`

[-] Create virtual environment
`python -m venv venv`

[-] Activate virtual environment
`cd venv/scripts`
`activate.bat`
`cd ../../`

[-] Run `python gemini.py`
